# Codigo Fuente del Sistema

Esta carpeta contiene el código fuente completo del sistema DML.

## Contenido

- **app.py** - Aplicación principal Flask (1143 líneas)
- **schema.sql** - Definición de la base de datos SQLite3
- **requirements.txt** - Dependencias Python necesarias
- **launcher.py** - Interfaz gráfica para ejecutar el sistema sin terminal
- **build_exe.py** - Script para compilar DML.exe
- **load_stock.py** - Script para cargar datos iniciales de stock
- **hash_password.py** - Herramienta para hashear contraseñas
- **show_stats.py** - Herramienta para mostrar estadísticas del sistema
- **compile_exe.py** - Script alternativo de compilación

## Ambiente Virtual

La carpeta `venv/` contiene el entorno virtual Python con todas las dependencias instaladas.

## Compilacion

Para compilar DML.exe ejecuta:
```
python build_exe.py
```

# Interfaz del Sistema

Esta carpeta contiene los archivos de interfaz web (HTML/CSS).

## Contenido

### Carpeta `templates/` 
Contiene 15 archivos HTML de templates Jinja2:

- **base.html** - Template base con navegación
- **index.html** - Página principal del sistema
- **login.html** - Página de autenticación
- **change_password.html** - Cambio de contraseña

**Módulo RAYPAC:**
- raypac_form.html - Formulario de entrada
- raypac_list.html - Lista de registros
- raypac_view.html - Detalle de registro

**Módulo DML:**
- dml_form.html - Formulario de entrada
- dml_list.html - Lista de fichas
- dml_view.html - Detalle de ficha
- dml_edit.html - Edición de ficha
- ficha_view.html - Vista de ficha

**Módulo STOCK:**
- stock_list.html - Lista de items
- stock_new.html - Nuevo item
- stock_edit.html - Editar item

**Módulo USUARIOS:**
- usuarios_list.html - Lista de usuarios
- usuario_form.html - Formulario de usuario
- usuario_edit.html - Editar usuario

### Carpeta `static/`
Contiene archivos estáticos:

- **style.css** - Estilos CSS de la aplicación (colores, layouts, responsivo)

## Estructura del Diseño

- Interfaz responsive (funciona en desktop, tablet y móvil)
- Diseño moderno con Bootstrap 5
- Colores corporativos: Azul (#1e3a8a) y Verde (#10b981)
- Navegación intuitiva con breadcrumbs
